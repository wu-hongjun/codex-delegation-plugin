CRITICAL DISAMBIGUATION
-----------------------
There are TWO distinct things called "workflow" in this repo:

1. **Claude Code native /workflows TUI panel** — a TUI-only interactive viewer
   opened by the `/workflows` slash command. Shows running workflow sessions
   with phase view, per-agent token usage, drill-down on agent tool calls + result.
   Controlled by keystrokes (p/x/r/s). Documented at code.claude.com/docs/en/workflows.

2. **cc workflow subcommand** — `node packages/plugin-codex/scripts/cc.mjs workflow`
   implemented at cc.mjs:309 as:
     promptTransformer: (p) => `ultracode: ${p}`
   This prepends `ultracode:` to the prompt, invoking the OMC `ultracode` skill
   (a single-session high-effort reasoning run), NOT the native /workflows panel.

This probe covers BOTH. The native /workflows TUI panel is what Plan 0015 asks about.

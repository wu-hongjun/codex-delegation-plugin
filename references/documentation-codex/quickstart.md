# Quickstart — Codex

> Source: https://developers.openai.com/codex/quickstart

Start using Codex in your IDE, CLI, or the cloud. Every ChatGPT plan includes Codex; you can also sign in with an OpenAI API key for API-billed usage.

## 1. App (recommended)

Available on macOS and Windows.

1. Download and install the Codex app.
   - macOS (Apple Silicon)
   - macOS (Intel)
   - Windows (Microsoft Store)
   - Linux: sign up for notifications
2. Open Codex and sign in with ChatGPT account **or** OpenAI API key.
3. Select a project folder.
4. Send your first message. Examples:
   - "Tell me about this project"
   - "Build a classic Snake game in this repo"
   - "Find and fix bugs in my codebase with minimal, high-confidence changes"

## 2. IDE Extension

Available for VS Code, Cursor, Windsurf, and VS Code Insiders.

1. Install the extension for your editor.
2. Open the Codex panel in the sidebar.
3. Sign in with ChatGPT or API key (starts in Agent mode).
4. Use Git checkpoints before/after tasks for easy reverting.

## 3. CLI

Supported on macOS, Windows, and Linux.

```bash
# macOS/Linux
curl -fsSL https://chatgpt.com/codex/install.sh | sh

# Windows
powershell -ExecutionPolicy ByPass -c "irm https://chatgpt.com/codex/install.ps1 | iex"

# npm
npm install -g @openai/codex

# Homebrew
brew install --cask codex
```

Run `codex` in a terminal, sign in, and start sending tasks.

## 4. Cloud

Access at [chatgpt.com/codex](https://chatgpt.com/codex).

1. Open Codex in the browser.
2. Set up the environment (connect a GitHub repository).
3. Launch coding tasks and monitor progress.
4. Review changes and create pull requests.

Getting started

Install v{{version}}, verify one provider, and run a complete job lifecycle.

Requirements

  • Codex CLI with plugin marketplace support.
  • Node.js 20 or later on PATH.
  • At least one local provider CLI: authenticated Claude Code (claude) or Google Antigravity (agy).

Normal plugin installation does not require an npm install or a repository clone.

Install from the public marketplace

codex plugin marketplace add https://github.com/wu-hongjun/codex-delegation-plugin
codex plugin add "delegate@codex-delegation-plugin"
codex plugin list

Confirm that the list shows delegate@codex-delegation-plugin as installed and enabled at version {{version}}. Restart Codex to load the new skill namespace.

Preflight a provider

Claude Code

$claude-setup
$claude-doctor --claude-access

Setup checks the common environment. Doctor is the deeper preflight for auth, model access, workspace, browser, and permission mode. An aggregate warn can still be ready when only an informational daemon probe warns; follow any explicit failed probe.

Google Antigravity

$agy-setup
$agy-doctor

Setup checks the binary and print-mode surface. Doctor adds targeted conversation-resume, workspace, and headless-permission checks. Neither runs a model task; authentication is checked during the first real delegation.

Run a first Claude Code job

$claude-delegate "Read the repository README and summarize the architecture. Do not edit files."

On the first delegation for this workspace and provider, review the privacy disclosure. The command returns a plugin job ID and a shorter provider session ID. Use the plugin job ID for the remaining commands. 

$claude-status --job <jobId> --json --compact
$claude-wait <jobId> --json --compact --timeout 5m
$claude-result <jobId>

If the job reaches awaiting_followup, the Claude session is still live. Continue it or stop it: 

$claude-followup <jobId> -- "List the three most important files for a new contributor."
$claude-result <jobId>
$claude-stop <jobId>

Run a first Antigravity job

$agy-delegate --mode plan -- "Read the repository README and summarize the architecture. Do not edit files."
$agy-wait <jobId> --json --compact --timeout 5m
$agy-result <jobId>
$agy-followup <jobId> -- "List the three most important files for a new contributor."
$agy-result <jobId>

The driver records this job's conversation UUID and the follow-up resumes that exact UUID. It does not use workspace-global agy --continue, so concurrent jobs remain isolated.

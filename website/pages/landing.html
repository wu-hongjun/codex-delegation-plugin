The operator loop

One command starts it. Five actions keep it accountable.

Start with a bounded instruction. Later Codex turns can wait, inspect partial or final output, continue a live Claude session, or stop work that no longer matters.

Architecture

A control plane, not a terminal scraper.

Runtime state comes from provider-native sessions and plugin-owned records. Session labels never masquerade as durable job identifiers.
